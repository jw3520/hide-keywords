# 🔍 Keyword Replacer (키워드 변환기)

사용자가 설정한 규칙에 따라 텍스트 내의 특정 키워드를 실시간으로 변환해주는 웹 도구입니다. 복잡한 텍스트 작업에서 반복적인 치환 과정을 자동화하고 효율화하기 위해 제작되었습니다.

## ✨ 주요 기능

- **실시간 텍스트 변환:** 입력창에 텍스트를 타이핑하는 즉시 설정된 규칙에 따라 결과가 출력됩니다.
- **다중 규칙 관리:** 여러 개의 변환 규칙을 동시에 적용할 수 있으며, 규칙을 자유롭게 추가하거나 초기화할 수 있습니다.
- **스왑(Swap) 기능:** '바꿀 키워드'와 '바뀔 키워드'를 단 한 번의 클릭으로 서로 맞바꿀 수 있습니다.
- **자동 키워드 생성:** 바꿀 키워드를 입력하면 바뀔 키워드에 자동으로 식별자(AAAA, BBBB...)가 생성되어 편리합니다.
- **정밀한 글자수 측정:** 
  - 입력창과 결과창의 글자수를 실시간으로 보여줍니다. (쉼표 포맷팅 적용)
  - 입력 제한(10,000자)을 시각적으로 안내하며, 초과 시 **강력한 빨간색 강조 효과**를 제공합니다.
- **원클릭 복사:** 변환된 결과를 버튼 하나로 클립보드에 복사할 수 있으며, 토스트 알림으로 성공 여부를 확인합니다.

---

## 🚀 사용 방법

1. **변환 규칙 설정:** 상단의 입력 칸에 '바꿀 키워드'와 '바뀔 키워드'를 입력합니다. (+ 버튼으로 규칙 추가 가능)
2. **텍스트 입력:** 좌측 '입력(Input)' 창에 본문을 입력합니다. (최대 10,000자)
3. **결과 확인:** 우측 '결과(Output)' 창에서 실시간으로 변환된 내용을 확인합니다.
4. **결과 복사:** 우측 상단의 복사 아이콘 버튼을 눌러 변환된 텍스트를 가져옵니다.
